# 课程表 DSL 批量导入格式说明

## 概述

本格式用于高校教务系统批量导出课程数据，支持各高校按统一规范提供课程信息，实现一键导入。

---

## 基础格式

```
课程名 @ 星期 节次 [w周次] [位置] [教师]
```

**分隔符说明：**

- `@` — 分隔课程名与排课信息（不可缺少）
- 空格 — 分隔各个字段

**字段说明：**

| 字段   | 必须 | 说明                                 | 示例       |
| ------ | ---- | ------------------------------------ | ---------- |
| 课程名 | 是   | 课程全称                             | 高等数学   |
| 星期   | 是   | 1-7，对应周一至周日                  | 1          |
| 节次   | 是   | 单节 `3` 或范围 `1-4`            | 1-2        |
| w周次  | 否   | 课程在哪一周出现，不写则所有周期显示 | w1,3,5     |
| 位置   | 否   | 上课地点                             | 教学楼A101 |
| 教师   | 否   | 授课教师姓名                         | 张老师     |

---

## 详细规则

### 1. 课程名

- 最长不超过 20 个字符
- 允许中文、英文、数字
- 不允许包含 `@` 字符

```
✅ 高等数学
✅ 大学英语 (II)
✅ CS101 程序设计
❌ 课程@名称  (不允许)
```

### 2. 星期 (1-7)

```
1 = 周一    5 = 周五
2 = 周二    6 = 周六
3 = 周三    7 = 周日
4 = 周四
```

### 3. 节次

支持两种格式：

**单节指定：**

```
3        → 第3节
```

**范围指定：**

```
1-2      → 第1节和第2节
3-5      → 第3、4、5节
```

> 每天最多 6 节（1-6）

### 4. 周次 `w`

表示课程在哪一周出现，使用 `w` 前缀，多个周次用逗号分隔。

**单周：**

```
w1        → 仅第1周显示
```

**多周列举：**

```
w1,3,5    → 第1、3、5周显示
```

**连续范围：**

```
w1-8      → 第1周到第8周显示
```

**留空（不写 w）：**

```
课程名 @ 1 1-2
```

不写周次表示该课程在**所有周期（所有周）**都显示。

> 注意：周次编号从 1 开始（用户视角的第1周），对应系统内部 cycleIndex 0。

# 基础课程

## 进阶用法

### 多节课同一课程

同一课程在不同时间段上课，分行书写：

```dsl
高等数学 @ 1 1-2 w1-16 教学楼A101 张老师
高等数学 @ 3 1-2 w1-16 教学楼A101 张老师
```

### 隔周课程

```dsl
体育理论 @ 2 1 w2,4,6,8 体育馆二楼 王老师
体育实践 @ 2 1 w1,3,5,7 操场        王老师
```

### 合班课程

```dsl
高等数学 (A班) @ 1 1-2 w1-16 教学楼A101 张老师
高等数学 (B班) @ 1 1-2 w1-16 教学楼A102 李老师
```

---

## 注意事项

1. **编码格式**：文件编码为 UTF-8
2. **注释行**：以 `#` 开头的行视为注释，自动忽略
3. **空行**：空行自动忽略
4. **顺序**：建议按星期顺序排列，便于人工核对
5. **周次上限**：系统支持最多 32 个周期，周次编号不要超过周期总数
6. **每行一条**：每行描述一节课，不要把多节课写在一行

---

## 错误处理

解析时遇到格式错误，该行会记录错误信息但不中断解析：

| 错误信息               | 原因               | 处理建议                    |
| ---------------------- | ------------------ | --------------------------- |
| `缺少 @ 分隔符`      | 课程名后缺少 `@` | 检查格式                    |
| `星期无效，应为 1-7` | 星期数字超出范围   | 修正为 1-7                  |
| `节次无效`           | 节次格式错误       | 使用 `3` 或 `1-4` 格式  |
| `周次格式无效`       | w后格式错误        | 使用 `w1,3,5` 或 `w1-8` |
