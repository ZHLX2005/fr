# 需求文档：课表换课数据结构

## 背景

课程存在"中途换课"场景：同一个时间段，前半段上 X 课程，结课后换成 Y 课程。

**示例**：
- 第 0-7 周期：高等数学
- 第 8-15 周期：线性代数（同一 slot 位置）

---

## 现状

**已实现**（不再需要改造）。当前数据层和过滤层都支持换课。

### 实际数据结构

```dart
// lib/core/timetable/domain/models.dart
class CourseItem {
  final List<int>? visibleInCycles; // null = 所有周期显示

  bool isVisibleInCycle(int cycleIndex) {
    if (visibleInCycles == null || visibleInCycles!.isEmpty) return true;
    return visibleInCycles!.contains(cycleIndex);
  }
}

// lib/core/timetable/presentation/timetable_store.dart
class TimetableState {
  /// 按 cellKey 索引，每个 key 对应该时间段的所有课程列表
  /// cellKey = 'd${dayOfCycle}_s$slotIndex'
  final Map<String, List<CourseItem>> items;
}
```

存储层（`hive_timetable_repository.dart`）用 `cellKey` 把同 (day, slot) 的多门课放进
`List<CourseItem>` 一并存：

```dart
// lib/core/timetable/data/hive_timetable_repository.dart:120
Future<void> saveItems(List<CourseItem> items) async {
  final grouped = <String, List<CourseItem>>{};
  for (final item in items) {
    grouped.putIfAbsent(item.cellKey, () => []).add(item);
  }
  for (final entry in grouped.entries) {
    await _itemsBox.put(entry.key, _courseListToJson(entry.value));
  }
}
```

过滤层（`cycleGridProvider`）按当前周期从同 cell 多门课里挑应显示的那门：

```dart
// lib/core/timetable/presentation/timetable_store.dart:349
static final cycleGridProvider =
    Provider.family<List<List<CourseItem?>>, int>((ref, cycleIndex) {
  ...
  for (final dayOfCycle in 0..daysPerCycle) {
    for (final slot in 0..slotsPerDay) {
      final courseList = state.items['d${dayOfCycle}_s$slot'] ?? [];
      for (final item in courseList) {
        if (item.isVisibleInCycle(cycleIndex)) {
          grid[dayOfCycle][slot] = item;
          break;
        }
      }
    }
  }
  ...
});
```

### 旧 → 新 迁移

仓储层在 `loadItems()` 里同时识别旧格式（单 `Map`）和新格式（`List<Map>`），
自动把旧的单课程项包装成 1 长度列表（`hive_timetable_repository.dart:97-114`）。
旧数据无需手动迁移。

---

## 效果

- 课程 A `visibleInCycles=[0,1,2,3,4,5,6,7]` → 周期 0-7 显示
- 课程 B `visibleInCycles=[8,9,10,11,12,13,14,15]` → 周期 8-15 显示
- 同一 (day, slot) 共存，周期不重叠则无冲突

### DSL 写法

```dsl
线性代数 @ 1 1-2 w1,2,3,4,5,6,7 教学楼A101 张老师
概率论   @ 1 1-2 w8,9,10,11,12,13,14,15 教学楼A101 李老师
```

（详见 `lib/core/timetable/DSL_FORMAT.md`）
