# 需求文档：课表换课数据结构

## 背景

课程存在"中途换课"场景：同一个时间段，前半段上 X 课程，结课后换成 Y 课程。

**示例**：
- 第 0-7 周期：高等数学
- 第 8-15 周期：线性代数（同一 slot 位置）

---

## 难点

当前 `allDaySlots[dayOfCycle][slotIndex] = CourseItem`（单一课程），无法同时存储两门课。

同一 (dayOfCycle, slotIndex) 位置上，数据层只能存一个 `CourseItem`，但换课场景需要存多个（课程 A 前半段 + 课程 B 后半段），且周期范围不重叠。

---

## 目的

让同一 slot 可以存储多门课程，各自通过 `visibleInCycles` 定义可见周期，达到"换课"效果。

---

## 当前设计

```dart
class CourseItem {
  final List<int>? visibleInCycles; // null = 所有周期显示

  bool isVisibleInCycle(int cycleIndex) {
    if (visibleInCycles == null || visibleInCycles!.isEmpty) return true;
    return visibleInCycles!.contains(cycleIndex);
  }
}

// 存储
allDaySlots[dayOfCycle][slotIndex] = CourseItem
```

---

## 期望方案

```dart
// 存储层：一个 slot 存多个课程
allDaySlots[dayOfCycle][slotIndex] = List<CourseItem>

// 过滤层（cycleGridProvider）：根据当前周期取对应课程
CourseItem? getCourse(int dayOfCycle, int slotIndex, int cycleIndex) {
  final courses = allDaySlots[dayOfCycle][slotIndex];
  // 取当前周期可见的课程
  return courses.firstWhere(
    (c) => c.isVisibleInCycle(cycleIndex),
    orElse: () => null,
  );
}
```

**效果**：
- 课程 A `visibleInCycles=[0,1,2,3,4,5,6,7]` → 周期 0-7 显示
- 课程 B `visibleInCycles=[8,9,10,11,12,13,14,15]` → 周期 8-15 显示
- 同一 slot 共存，周期不重叠则无冲突
