### 墙壁放置合法性验证

`GameModel.swift:61-78` 的 `iWallIsAllow` 方法：

```swift
// 来源: GameModel.swift:61-78
func iWallIsAllow(data: DataModel) -> Bool {
    guard data.x >= 0 && data.y >= 0 else { return false }
    guard data.x < 9 && data.y < 9 else { return false }

    for id in data.wallIds {
        if gameWalls[id] { return false }  // 与现有墙壁重叠
    }

    removeNearLink(data)
    if GameAi.pathForPlayer(true).count > 0 {
        if GameAi.pathForPlayer(false).count > 0 {
            addNearLink(data)
            return true    // 双方都有通往终点的路径
        }
    }
    addNearLink(data)
    return false  // 会堵死某方路径
}
```

核心条件：
1. 坐标在 0-8 范围内
2. 不与任何已放置的墙壁重叠
3. **最关键**：双方都必须至少有一条通往终点的路径（通过 BFS 验证）

## 跳跃规则

`GameModel+Logic.swift:6-20` 的 `scopeForPlayer` 方法：

```swift
// 来源: GameModel+Logic.swift:6-20
func scopeForPlayer(player: Int, rival: Int) -> [Int] {
    var nears = [Int]()
    for near in gameNears[player] {
        if near != rival {
            nears.append(near)           // 正常邻接点
        } else {
            for rivalNear in gameNears[rival] {
                if rivalNear != player {
                    nears.append(rivalNear)  // 跳过对方棋子
                }
            }
        }
    }
    return nears
}
```

当棋子相邻时（`near == rival`），会额外将对方棋子**后方的格子**也加入可移动范围，实现"跳过"效果。

## 游戏结束判定

`GameModel.swift:123-139`：

```swift

var status: GameStatus {
```
