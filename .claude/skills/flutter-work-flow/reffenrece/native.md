和原生功能的交互功能规范

和安卓原生的沟通存放到home_widget当中 进行统一的状态管理和维护


设计原则总结



  ┌────────────┬──────────┬───────────────────────┬──────────┐

  │    目录    │   职责   │       依赖规则        │ 可测试性 │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ core/      │ 基础配置 │ 不依赖任何业务层      │ ⭐⭐⭐   │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ models/    │ 数据模型 │ 只依赖json_annotation │ ⭐⭐⭐   │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ utils/     │ 工具函数 │ 纯函数，无状态        │ ⭐⭐⭐   │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ services/  │ 业务服务 │ 依赖models, core      │ ⭐⭐     │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ providers/ │ 状态管理 │ 依赖models, services  │ ⭐⭐     │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ widgets/   │ 通用组件 │ 可依赖providers       │ ⭐⭐     │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ screens/   │ 页面UI   │ 可依赖所有层          │ ⭐       │

  ├────────────┼──────────┼───────────────────────┼──────────┤

  │ routes/    │ 路由管理 │ 依赖screens           │ ⭐⭐     │

  └────────────┴──────────┴───────────────────────┴──────────┘
