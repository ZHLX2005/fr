血条显示动画逻辑

当前方案A参数：dotTop=60, 右侧独立区域

dotTop（圆点距顶） 60px
barHeight（血条总长） 100px （固定）
barX（血条位置） 右侧独立区域
barWidth（线宽） 1px
Miss 时血条变短动画流程

1. Miss 前：health = 100%

100% 血量
fillHeight = 100px
fillHeight = barHeight × health = 100 × 1.0 = 100px
细线从底部(lineBottom)向上填充100px，满血。

2. Miss -15%：health = 85%，动画过渡

动画过渡
100px → 85px (animating...)
fillHeight = barHeight × health = 100 × 0.85 = 85px
fillHeight 从 100px 动画过渡到 85px（用 AnimationController，duration=300ms）

3. Miss 后：health = 85%，细线缩短

85% 血量
fillHeight = 85px
细线底部(lineBottom)不变，圆点(dotTop)不变，
填充高度从100px缩短到85px。视觉上血条从底部向上收缩。

4. Perfect +5%：health = 90%，动画上升

动画上升
85px → 90px (animating...)
fillHeight = 100 × 0.90 = 90px
细线从85px动画扩展到90px。血量回升时血条从底部向上增长。

核心实现逻辑

// 固定参数（不随屏幕变化） final dotTop = 60.0; // 圆点距顶60px final barHeight = 100.0; // 固定血条总长100px final barX = w - 12.0; // 右侧独立区域，距右边缘12px // 绘制：圆点 + 细线背景 + 细线填充（从底部向上） void _paintHealthBar(Canvas canvas) { // 顶部圆点（固定位置） canvas.drawCircle(Offset(barX, dotTop), 3.0, dotPaint); // 细线背景（固定100px） canvas.drawLine( Offset(barX, dotTop + 4), Offset(barX, dotTop + 4 + barHeight), lineBgPaint, ); // 细线填充 = 血量百分比 × 固定高度 final fillHeight = barHeight * health.clamp(0.0, 1.0); // 从 barHeight 动画到 fillHeight，duration=300ms canvas.drawLine( Offset(barX, dotTop + 4 + barHeight), Offset(barX, dotTop + 4 + barHeight - animatedFillHeight), lineFillPaint, ); }
